# Examples

## minimal-artifact

A simplified ARA artifact for "Attention Is All You Need" showing the basic structure:
- `PAPER.md` with YAML frontmatter and layer index
- `logic/claims.md` with falsifiable claims and proof references
- `logic/problem.md` with observations, gaps, and key insight
- `trace/exploration_tree.yaml` with a minimal research DAG

A real artifact generated by the compiler would include all mandatory files (15+), 8+ exploration tree nodes, 5+ concepts, 3+ experiment plans, evidence tables/figures, and code stubs.

To generate a full artifact, install the compiler skill and run:

```
/compiler path/to/paper.pdf
```

## ResNet demo

This repo now also includes a concrete ResNet-oriented demo flow:

- `resnet-paper.pdf`: the original ResNet paper PDF
- `resnet-ara-example/`: a worked ARA-style artifact for the degradation-to-residual-learning slice
- `resnet-walkthrough.md`: how to use the compiler workflow on this example

Suggested order:

1. Read `resnet-walkthrough.md`
2. Open `resnet-paper.pdf`
3. Compare the paper against `resnet-ara-example/`
