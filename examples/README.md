# Examples

## minimal-artifact

A simplified ARA artifact for "Attention Is All You Need" showing the basic structure:
- `PAPER.md` with YAML frontmatter and layer index
- `logic/claims.md` with falsifiable claims and proof references
- `logic/problem.md` with observations, gaps, and key insight
- `trace/exploration_tree.yaml` with a minimal research DAG

A real artifact generated by the ingestor would include all mandatory files (15+), 8+ exploration tree nodes, 5+ concepts, 3+ experiment plans, evidence tables/figures, and code stubs.

To generate a full artifact, install the ingestor skill and run:

```
/ingestor path/to/paper.pdf
```
