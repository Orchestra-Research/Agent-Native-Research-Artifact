# ResNet Skill Demo

This folder is a concrete test run for the `ingestor` workflow using the ResNet paper.

## Layout

- `input/resnet-paper.pdf`: source paper input
- `output/ara/`: generated ARA-style artifact

## Intent

This is not the output of a packaged CLI binary. It is a manual simulation of what the `ingestor` skill is supposed to produce when given the ResNet paper as input.

## Quick path

1. Open `input/resnet-paper.pdf`
2. Read `output/ara/PAPER.md`
3. Check `output/ara/logic/claims.md`
4. Check `output/ara/evidence/tables/`
5. Check `output/ara/trace/exploration_tree.yaml`
